# PickMovieTonight

![Status](https://img.shields.io/badge/status-active-2ea44f)

PickMovieTonight is a lightweight movie discovery web application that helps users quickly find something to watch based on mood, pacing, energy, and overall vibe.

The project focuses on fast, distraction-free movie discovery with curated recommendations, interactive browsing, and integrated trailers.

---

## Features

### Discover Movies

- Interactive movie recommendation wizard
- Curated movie recommendations
- Browse by decade
- Random movie discovery
- Embedded YouTube trailers

### User Experience

- Responsive mobile-first design
- Lightweight static frontend
- Fast loading
- Recently watched trailer history
- Clean, distraction-free interface

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- TMDb API
- YouTube embeds
- Netlify
- Git & GitHub

---

## Project Goals

PickMovieTonight was created to make movie selection fast and enjoyable.

Instead of scrolling endlessly through streaming services, users receive curated recommendations and can immediately watch trailers to decide what to watch.

---

## Running Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/pickmovietonight.git
```

Serve the project using any static web server.
