# PickMovieTonight

PickMovieTonight is a lightweight movie discovery tool designed to help users quickly find something worth watching based on mood, pacing, energy, and vibe.

## Features

- Interactive movie picker wizard
- Curated movie dataset
- Random trailer discovery
- Browse movies by era
- Responsive mobile-first design
- Lightweight static frontend

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- Netlify
- TMDb API
- YouTube embeds

## Live Site

https://pickmovietonight.com

## Notes

Movie posters, trailers, and related metadata may come from third-party providers such as TMDb and YouTube.
